## 项目概述

本项目（编号：python027）是一个网络应用程序，专注于高校人才招聘和校园求职服务，通过招聘数据洞察，实现校企人才对接。

### 前端技术

- Html
- Css
- Js
- Vue2.0
- Element-ui

### 后端技术

- Python3.7
- Flask

- 开发工具：PyCharm、VSCODE

- 数据库管理工具：Navicat 10

- 浏览器：谷歌浏览器
